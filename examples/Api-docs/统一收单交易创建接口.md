# 统一收单交易创建接口

## 功能说明

**接口名称**：`alipay.trade.create`

商户通过该接口进行交易的创建下单

## 请求

- **URL**：`/v3/alipay/trade/create`
- **Method**：`POST`
- **Content-Type**：`application/json`
